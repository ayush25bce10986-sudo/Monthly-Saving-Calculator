# MONTHLY SAVINGS CALCULATOR

##  Project Title

**Monthly Savings Calculator**

## ðŸ“˜ Overview of the Project

The Monthly Savings Calculator is a simple Python-based program that
helps users calculate their total monthly expenses, monthly savings, and
savings rate.\
If the user is saving money, the program also includes a **Savings Goal
Calculator** that estimates the time required to reach a financial goal.

This tool is ideal for students, beginners, and anyone learning Python
basics like variables, user input, arithmetic operations, and
conditional statements.

------------------------------------------------------------------------

##  Features

-   Takes monthly income and expense inputs from the user.
-   Calculates:
    -   Total monthly expenses\
    -   Monthly savings\
    -   Savings percentage\
-   Identifies if the user is:
    -   Saving money\
    -   Breaking even\
    -   Overspending\
-   Savings Goal Calculator:
    -   User enters a financial goal (e.g., car, vacation)
    -   Program calculates how many months/years are required to achieve
        the goal.
-   Easy-to-read formatted output.

##  Steps to Install & Run the Project

### **1. Install Python**

Download and install Python from:\
https://www.python.org/downloads/

### **2. Create a project folder**

    mkdir savings_calculator
    cd savings_calculator

### **3. Create a Python file**

Create a file named:

    savings_calculator.py

### **4. Paste the full program code into the Python file**

### **5. Run the program**

In terminal or command prompt:

    python savings_calculator.py
